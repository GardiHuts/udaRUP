options

Propiedades de configuración del componente.

Name Type Argument Default Description
submitButton Selector

Identificador del botón de envío del formulario. Sirve para que dicho botón solo se muestre en el último paso del asistente.
submitFnc jQuery.rup_validate~onSubmitFnc <optional>

Función que se ejecuta previamente al envío del formulario.
summary boolean <optional>
 true

Indica si se debe generar o no el paso resumen. Este paso mostrará (como texto plano) los valores seleccionados en los diferentes pasos habilitados. Si un paso se encuentra deshabilitado no se utilizarán sus valores para la generación del resumen. El paso de resumen se genera cuando se navega hacia él.
summaryWithAccordion boolean <optional>
 false

Indica si el paso resumen debe formatear los diferentes pasos del asistente mediante elementos que usan el componente accordion.
summaryWithAccordionSpaceBefore boolean <optional>
 true

Indica si cuando se genera un resumen con los diferentes pasos presentados con accordion, se debe dejar un espacio (
) entre el contenido y la barra con los pasos.
summaryWithAccordionSpaceAfter boolean <optional>
 true

Indica si cuando se genera un resumen con los diferentes pasos presentados con accordion, se debe dejar un espacio (
) entre el contenido y el contenedor de botones (siguiente y enviar).
summaryTabs2Accordion boolean <optional>
 false

Indica si los componentes rup_tab de los diferentes pasos del wizard se deben convertir en elementos del componente accordion a la hora de generar el paso resumen.
summaryFnc_PRE jQuery.rup_validate~onSummaryFnc_PRE <optional>

Función que se invocará previamente a la generación del paso resumen. En caso de devolver false no se generaría el resumen.
summaryFnc_INTER jQuery.rup_validate~onSummaryFnc_INTER <optional>

Función que se ejecuta una vez ha comenzado la generación del paso resumen. Se habrá generado la capa contenedora y duplicado el contenido de los pasos anteriores pero sin llegar a procesarse (cambiarse por texto plano). En caso de devolver false no se continuaría con el procesado del contenido de los pasos.
summaryFnc_POST jQuery.rup_validate~onSummaryFnc_POST <optional>

Función que se ejecuta una vez se ha generado el paso resumen.
stepFnc object <optional>

Objeto json que contiene las diferentes funciones a ejecutar al navegar hacia cada uno de los pasos. En caso de devolver false no se continuaría la navegación hacia dicho paso. La clave de cada elemento será el número del paso y el valor será la función a ejecutar.
disabled Array.<Integer> <optional>

Array que indica los pasos a deshabilitar al inicio. En caso de que el elemento del array sea un número (numeric) se deshabilitará dicho paso y en caso de que sea un literal (string) se procesará como un intervalo que deberá definirse como “X-Y”.
accordion object <optional>

Define de forma general el funcionamiento del componente rup_accordion en el paso de resumen.
rupAccordion object <optional>

Define el funcionamiento del patrón rup_accordion en el resumen (de los objetos rup_accordion existentes en los pasos anteriores). No tiene valor por defecto, por lo que en caso de no definirse se toma el valor del parámetro accordion.
summaryAccordion object <optional>

Define el funcionamiento del patrón rup_accordion en el resumen para cada uno de los pasos que lo componen (si configura la generación de resumen y conversión de pasos en accordion). No tiene valor por defecto, por lo que en caso de no definirse se toma el valor del parámetro accordion.
tabAccordion object <optional>

Define el funcionamiento del patrón rup_accordion en el resumen, para cada los objetos rup_tab existentes en los pasos anteriores (si configura la generación de resumen y conversión de pestañas en accordion). No tiene valor por defecto, por lo que en caso de no definirse se toma el valor del parámetro accordion.
rupTabsElement string <optional>
 <h4/>

Define el funcionamiento del patrón rup_accordion en el resumen, para cada los objetos rup_tab existentes en los pasos anteriores (si configura la generación de resumen y conversión de pestañas en accordion). No tiene valor por defecto, por lo que en caso de no definirse se toma el valor del parámetro accordion.
